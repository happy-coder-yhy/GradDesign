# 可搜索下拉列表功能说明

## 功能介绍

为了方便用户快速选择节点，系统现在使用 **Element Plus** 的可搜索下拉列表组件，支持：

- 🔍 **实时搜索过滤**：在输入框中输入节点ID即可快速过滤
- 📂 **分组显示**：节点按类型分组（机位点、跑道点）
- ✨ **一键清除**：支持快速清除已选择的节点
- 🎨 **深色主题**：完美融入系统的深色UI风格

## 使用方法

### 1. 选择起点
1. 点击"🟢 选择起点"下拉框
2. 直接滚动查找或输入节点ID进行搜索
3. 例如：输入 "5" 可以快速找到 "机位点 #5"
4. 点击选择节点

### 2. 选择终点
1. 点击"🟠 选择终点"下拉框
2. 同样可以通过输入节点ID快速搜索
3. 点击选择目标节点

### 3. 计算路径
1. 同时选择起点和终点后，"🚀 计算自定义路径"按钮会自动启用
2. 点击按钮即可计算A*优化路径

### 4. 清除选择
- 点击下拉框右侧的 ❌ 图标可清除单个选择
- 点击"清除选择"按钮可同时清除起点和终点

## 搜索示例

| 输入内容 | 匹配结果 |
|---------|---------|
| `1` | 机位点 #1, 跑道点 #1 |
| `10` | 机位点 #10, 跑道点 #10 |
| `机位` | 所有包含"机位"的选项 |
| `跑道` | 所有包含"跑道"的选项 |

## 技术实现

- **组件库**: Element Plus
- **组件**: `el-select` + `el-option-group`
- **关键属性**:
  - `filterable`: 开启搜索功能
  - `clearable`: 支持清除
  - `placeholder`: 占位提示文本
  - `@change`: 值变化时触发

## 样式定制

所有Element Plus组件都已定制为深色主题，与系统整体风格保持一致：
- 深蓝背景色
- 青蓝色高亮
- 半透明毛玻璃效果
- 流畅的过渡动画
